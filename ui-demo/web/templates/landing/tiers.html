Tiered Packages

What do you need?

Each tier builds on the previous. Select based on your deployment scope and institutional capacity.

Tier 1
Foundation
Single use case, single issuer, basic issuance and verification. Ideal for pilots and proof-of-concept.
  • ✓ Credential issuance (single + batch)
  • ✓ Basic verification portal
  • ✓ Web eLocker for holders
  • ✓ Single schema/template
  • ✓ QR + PDF output
  • — Mobile wallet
  • — Multi-issuer onboarding
  • — Selective disclosure
Tier 2
Ecosystem
Multiple use cases, self-service issuer onboarding, mobile wallet, cross-sector deployment.
  • ✓ Everything in Foundation
  • ✓ Self-service issuer onboarding
  • ✓ Mobile wallet (Inji or compliant)
  • ✓ Multiple schemas/templates
  • ✓ Selective disclosure (SD-JWT)
  • ✓ Trust registry
  • ✓ Analytics dashboards
  • — Cross-DPG interoperability
Tier 3
National Scale
Full interoperability, multi-DPG bridging, last-mile channels, ZKP, agent-assisted flows.
  • ✓ Everything in Ecosystem
  • ✓ Cross-DPG adaptor layer
  • ✓ Last-mile channels (SMS, USSD, IVR)
  • ✓ ZKP / AnonCreds support
  • ✓ Agent-assisted verification
  • ✓ Offline verification
  • ✓ Multi-tenancy
  • ✓ Schema harmonization tools
